# Frequently Asked Questions

## General

**Q: Is this information current?**
A: No guarantees. Always verify with organizations directly.

**Q: Can I use this on my website?**
A: Yes! It's MIT licensed—use it anywhere, even commercially.

**Q: Will you add my city?**
A: Maybe. Open an Issue and we'll discuss.

**Q: What should I do if I find outdated information?**
A: Open an Issue using the [Outdated Resource](../../issues/new) template with the details.

## Technical

**Q: Do I need to install anything?**
A: No. It's a single HTML file with no dependencies.

**Q: Does it work on mobile?**
A: Yes, it's fully responsive.

**Q: Can I customize the colors?**
A: Yes, edit the CSS variables at the top of the file. See [CUSTOMIZATION.md](CUSTOMIZATION.md).

**Q: Can I embed this on my website?**
A: Yes, use an iframe. See [README.md](README.md) for examples.

## Contributing

**Q: How do I suggest a new resource?**
A: Open an Issue or submit a Pull Request with your suggestion.

**Q: Will my contribution be accepted?**
A: No guarantees, but all thoughtful contributions are appreciated.

**Q: How long until my PR is reviewed?**
A: This is a volunteer project—no timeline commitments.

**Q: What if my PR is rejected?**
A: We appreciate all contributions. Feedback will be provided on why it wasn't merged.

## Privacy & Security

**Q: Does this widget track me?**
A: No. It collects zero analytics, cookies, or personal data.

**Q: Is the widget accessible?**
A: Yes, it meets WCAG 2.1 AA accessibility standards and is fully keyboard navigable.

**Q: Is there a responsible disclosure policy?**
A: Yes, see [SECURITY.md](SECURITY.md) for reporting vulnerabilities.
