## [1.0.0] - 2025-11-30

### Added
- Initial release
- Portland, Seattle, and Spokane resource directories
- Categories: Housing, Food, LGBTQ+, Domestic Violence, Child Abuse, Sexual Assault
- Tab-based city navigation
- Accordion-style category display
- Glassmorphic design
- Full keyboard accessibility
- Mobile responsive layout
- Comprehensive documentation

### Known Issues
- Resources may be outdated (see disclaimer)
- No automatic link checking
